## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd flowlaunch
```

Install dependencies

```bash
  npm install
  "@headlessui/react": "^2.0.4",
  "@smastrom/react-rating": "^1.5.0",
  "axios": "^1.7.2",
  "next": "14.2.3",
  "react": "^18",
  "react-dom": "^18",
  "react-icons": "^5.2.1"
```

Install devDependencies

```bash
  "daisyui": "^4.11.1",
  "eslint": "^8",
  "eslint-config-next": "14.2.3",
  "postcss": "^8",
  "tailwindcss": "^3.4.1"
```

Start the server

```bash
  http://localhost:3000
```
